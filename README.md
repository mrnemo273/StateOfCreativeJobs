# State of Creative Jobs

A labor market research tool tracking the health of 20 creative and design job titles over time — monitoring demand, compensation trends, AI displacement risk, and practitioner sentiment.

Built with Next.js 14, TypeScript, Tailwind CSS, and Recharts.

## Documentation

All project documentation lives in `/docs`:

- **[GOALS.md](docs/GOALS.md)** — Project vision, mission, and guiding principles for AI collaborators
- **[PRD.md](docs/PRD.md)** — Product Requirements Document: what the product does, who it's for, and how it works
- **[TRD.md](docs/TRD.md)** — Technical Requirements Document: architecture, file structure, APIs, and implementation details
- **[DESIGN_SYSTEM.md](docs/DESIGN_SYSTEM.md)** — Visual design specification: typography, color, grid, and component rules

Historical phase specs that guided development are archived in `docs/archive/`.

## Quick Start

```bash
npm install
npm run dev
```

## For AI Collaborators

If you're an AI model continuing work on this project, start by reading the docs in this order:

1. `docs/GOALS.md` — understand why this exists and the build philosophy
2. `docs/PRD.md` — understand what the product does
3. `docs/TRD.md` — understand how it's built and what not to break
4. `docs/DESIGN_SYSTEM.md` — understand the visual rules (they're strict)
